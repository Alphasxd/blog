# Shell 常用快捷键

- Tab 命令补全
- Ctrl
  - a - 光标移动到行首
  - e - 光标移动到行尾
  - w - 剪切光标之前的单词，作用等同于删除光标之前的单词
  - u - 剪切光标之前的所有内容，作用等同于清空光标所在行
  - k - 剪切光标及其后面的所有内容
  - y - 粘贴剪切板的内容到光标之前
  - l - 清屏
  - r - 在历史命令中搜索包含指定字符串的命令
    - 再按Ctrl + r 查找上一个
    - 按 `→` 右箭头选择当前命令到 shell  
    - 按 `⏎` 回车选择并执行当前命令
- `↑` 和 `↓` 上下箭头分别搜索历史命令
- PgUp 从后往前搜索历史命令，写一段字符串后按PgUp键，事先搜索以当前字符串开头的命令
